# 02-questionnaires

Ce module produit et suit les questionnaires d’instruction à partir du cadrage.

## Entrées

- fiche de cadrage dynamique ;
- questions de contrôle ;
- carte des risques ;
- pièces déjà disponibles ;
- lacunes documentaires ;
- réponses déjà reçues.

## Sorties

- questionnaire n° 1 ;
- questionnaires complémentaires ;
- table de rattachement questions / risques / pièces attendues ;
- suivi des réponses ;
- liste des relances ou clarifications nécessaires.

## Règle

Le questionnaire ne doit pas être une demande documentaire exhaustive. Chaque question doit être justifiée par un risque, une question de contrôle, une hypothèse ou une pièce nécessaire à la preuve.
