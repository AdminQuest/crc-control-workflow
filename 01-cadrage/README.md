# 01-cadrage

Ce module transforme les éléments d’ouverture du contrôle en stratégie d’instruction.

## Entrées

- lettre d’ouverture ;
- fiche de programmation ;
- précédent ROD et suivi des recommandations ;
- premiers éléments financiers ;
- dossier permanent ;
- contexte institutionnel ;
- entretien d’ouverture.

## Sorties

- fiche de cadrage dynamique ;
- carte des risques ;
- questions de contrôle ;
- hypothèses à tester ;
- trame d’entretien d’ouverture ;
- liste des pièces attendues ;
- base du questionnaire n° 1.

## Règle

Chaque question posée ultérieurement doit pouvoir être rattachée à une question de contrôle, à un risque ou à une hypothèse explicitement formulée.
