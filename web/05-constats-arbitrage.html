05 — Constats-arbitrage

Décider ce qui mérite d’être porté au rapport.

AccueilDocuments maîtresConstatsPrompt-engineTest T01

Règle

Un constat ne doit être porté au rapport que s’il est significatif, prouvé, illustrable et domicilié dans une seule partie.

Prompt — extraire les constats délibérables

Tu agis comme assistant d’arbitrage des constats pour un contrôle CRC. Tu ne rédiges pas le rapport. Tu sélectionnes les constats susceptibles d’être portés au RIOP.

À partir du document maître ou de la fiche d’analyse fournie, produis une matrice de constats délibérables.

Pour chaque constat : formule le constat en une phrase ; rattache-le à une question de contrôle ; indique la preuve principale ; indique l’illustration obligatoire dans le corps du rapport ; qualifie la portée ; qualifie la significativité ; mentionne les limites probatoires ; mentionne les arguments contraires connus ; propose une domiciliation unique ; propose une décision : garder, réduire, déplacer, annexer, compléter, abandonner.

Règles : écarte les éléments purement descriptifs ; ne retiens pas un constat non prouvé ; ne retiens pas un constat significatif sans illustration possible ; ne confonds pas fait établi, déclaration et analyse ; signale les constats qui relèvent d’une note séparée ou d’une suite sensible.
