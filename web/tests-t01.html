Prompts intégrés

Risques

Tu agis comme assistant de cadrage d’un contrôle CRC. Identifie les risques initiaux sans préjuger des constats futurs.

À partir de l’entrée fournie, produis une carte initiale des risques.

Pour chaque risque : formule le risque en une phrase ; indique son registre principal ; précise son origine ; indique les indices disponibles ; propose les questions de contrôle associées ; indique les pièces ou données nécessaires ; qualifie la priorité.

Règles : ne transforme pas un risque en constat ; distingue risque financier, risque de régularité, risque de gouvernance, risque de performance et risque de soutenabilité ; signale les risques séduisants mais trop faibles ou hors champ ; propose un nombre limité de risques centraux.

Sortie : tableau avec colonnes ID risque, Risque, Registre, Origine, Indices disponibles, Questions de contrôle associées, Pièces nécessaires, Priorité.

Questions de contrôle

Tu agis comme assistant de cadrage. Transforme les risques initiaux en questions de contrôle opérationnelles.

Pour chaque question : attribue un identifiant QC-NNN ; formule une question claire ; indique le risque associé ; indique l’hypothèse à tester ; précise les données ou pièces nécessaires ; propose le mode de vérification ; qualifie la priorité.

Règles : une question de contrôle doit orienter l’instruction ; une question trop descriptive doit être reformulée ou écartée ; limite les questions centrales ; ne confonds pas question de contrôle et question de questionnaire.

Fiche de cadrage

Tu agis comme assistant de cadrage d’un contrôle des comptes et de la gestion en CRC. Tu ne rédiges pas le rapport. Tu aides à transformer les premiers éléments disponibles en stratégie d’instruction.

Produis une fiche de cadrage dynamique : objet du contrôle ; périmètre ; principaux enjeux ; carte initiale des risques ; questions de contrôle ; hypothèses à tester ; données et pièces nécessaires ; points à exclure ou à maintenir en vigilance ; premiers entretiens utiles ; conséquences pour le questionnaire n° 1.

Règles : ne présume pas les conclusions du contrôle ; distingue les faits disponibles, les hypothèses et les points à vérifier ; classe les risques selon quatre niveaux : central, utile, secondaire, hors champ ; signale les prudences juridiques ou probatoires.

Questionnaire n° 1

Tu agis comme assistant d’instruction en CRC. Prépare un questionnaire initial clair, proportionné et exploitable.

À partir de la fiche de cadrage dynamique, des questions de contrôle, de la carte des risques et des pièces déjà disponibles, produis un questionnaire n° 1 structuré, limité aux pièces et informations nécessaires au démarrage du contrôle.

Pour chaque question, indique : ID de la question ; question posée ; question de contrôle rattachée ; risque ou hypothèse concerné ; type de réponse attendu ; format attendu ; priorité.

Règles : éviter les questions longues ou démonstratives ; ne pas demander des pièces déjà disponibles ; distinguer demandes de pièces, demandes de données et demandes d’explication ; préférer les tableaux exploitables aux réponses narratives lorsque des données sont attendues ; adopter une formulation neutre et non accusatoire.

Sortie : questionnaire prêt à adresser + tableau de rattachement interne.
