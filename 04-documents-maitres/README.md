# 04-documents-maitres

Ce module transforme les atomes et registres en analyses thématiques.

## Entrées

- atomes de preuve ;
- registres ;
- questions de contrôle ;
- sources classées ;
- réponses aux questionnaires.

## Sorties

- documents maîtres ;
- fiches d’analyse financière ;
- fiches d’analyse juridique ;
- fiches commande publique ;
- fiches gouvernance ;
- fiches patrimoine ;
- fiches risques ;
- liste des pièces manquantes.

## Règle

Le document maître peut être large, mais il doit distinguer les faits établis, les déclarations, l’analyse, les limites probatoires et les pistes de constat.
