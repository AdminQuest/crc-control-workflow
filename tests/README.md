# Tests fonctionnels

Ce dossier sert à tester les chaînes de travail du dépôt avec des cas d’entrée simples.

## Test disponible

```text
T01-cadrage-questionnaire
```

Objectif : partir d’une lettre d’ouverture ou d’un descriptif de contrôle, puis produire :

1. une carte initiale des risques ;
2. des questions de contrôle ;
3. une fiche de cadrage dynamique ;
4. un questionnaire n° 1 rattaché aux questions de contrôle.

## Règle de test

Le test ne doit pas chercher à reconstituer toute l’instruction. Il vérifie que le workflow produit une stratégie d’instruction cohérente, traçable et réutilisable.
