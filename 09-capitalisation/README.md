# 09-capitalisation

Ce module conserve les enseignements tirés des contrôles.

## Entrées

- retours de délibéré ;
- apostilles ;
- modifications demandées ;
- observations abandonnées ;
- recommandations conservées ou reformulées ;
- réponses contradictoires ;
- ROD publiés ;
- retours internes.

## Sorties

- bibliothèque de formulations robustes ;
- erreurs fréquentes ;
- recommandations types ;
- points de vigilance ;
- retours d’expérience ;
- prompts améliorés.

## Règle

La capitalisation ne doit pas transformer des cas particuliers en règles générales sans validation. Elle conserve les enseignements, leur contexte et leurs limites.
