# Taxonomie des registres

Les registres permettent de classer les atomes de preuve et les constats selon leur fonction dans le contrôle.

## Registres principaux

| Registre | Usage |
|---|---|
| `finances` | Situation financière, budget, dette, CAF, fonds de roulement, trésorerie |
| `regularite` | Respect des règles applicables, procédures, actes, compétences |
| `gouvernance` | Organisation institutionnelle, instances, contrôle interne, décision |
| `performance` | Efficience, efficacité, qualité de gestion, résultats |
| `strategie` | Orientations, programmation, trajectoire, priorisation |
| `commande_publique` | Marchés, concessions, avenants, mise en concurrence |
| `rh` | Effectifs, masse salariale, organisation du travail, rémunérations |
| `patrimoine` | Actifs, inventaire, maintenance, renouvellement, immobilier |
| `risques` | Contentieux, assurabilité, cybersécurité, continuité, hors bilan |
| `satellites` | SPL, SEM, associations, délégataires, organismes liés |
| `deontologie` | Conflits d’intérêts, déclarations, déports, probité |
| `donnees` | Bases, fichiers, extractions, séries, tableaux |
| `entretiens` | Déclarations recueillies en entretien ou réunion |
| `textes` | Normes applicables, jurisprudence, doctrine |
| `procedure` | Actes de procédure du contrôle, communicabilité, contradiction |

## Règle

Un atome peut relever de plusieurs registres. Un constat délibérable doit en revanche être domicilié dans un registre principal pour éviter les redites.
