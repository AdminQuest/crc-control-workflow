# 08-contradiction-riod

Ce module traite la contradiction sans refaire l’instruction.

## Entrées

- ROP ;
- réponses contradictoires ;
- pièces nouvelles ;
- extraits transmis aux tiers ;
- apostilles du premier délibéré.

## Sorties

- matrice de contradiction ;
- qualification des arguments ;
- effet utile de chaque réponse ;
- propositions de maintien, modification ou abandon ;
- RIOD simplifié ;
- tableau ROP / ROD ;
- actualisation des recommandations.

## Règle

La contradiction ne sert pas à pallier les insuffisances de l’instruction. Elle sert à apprécier l’effet utile des réponses reçues sur les observations provisoires.
