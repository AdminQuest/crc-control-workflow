# 05-constats-arbitrage

Ce module décide ce qui mérite d’être porté dans le rapport.

## Entrées

- documents maîtres ;
- fiches d’analyse ;
- atomes de preuve ;
- registres ;
- limites probatoires identifiées.

## Sorties

- fiches de constats délibérables ;
- matrice de significativité ;
- matrice de domiciliation ;
- liste des constats à abandonner ;
- liste des constats à compléter ;
- pré-plan du rapport.

## Règle

Un constat ne doit être porté au rapport que s’il est significatif, prouvé, illustrable et domicilié dans une seule partie.
