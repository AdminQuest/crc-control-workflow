# 07-qualite-delibere

Ce module éprouve le projet avant dépôt et avant délibéré.

## Entrées

- projet de RIOP ou de ROP ;
- matrice de constats ;
- recommandations ;
- apostilles ;
- DLRé ;
- paquets de rédaction.

## Sorties

- audit anti-redites ;
- audit preuve / DLRé ;
- audit des illustrations ;
- audit des recommandations ;
- simulation de pré-délibéré ;
- points à arbitrer.

## Règle

Ce module ne cherche pas à améliorer le style. Il teste la robustesse, la proportion, la lisibilité et la capacité du rapport à tenir en délibéré.
